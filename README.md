# 🏠 GharKhoj – Find Your Perfect Home Easily

**GharKhoj** is a modern property listing application that connects **property owners** and **home seekers**.  
Built with **Flutter** and **Node.js**, it provides a seamless experience to search, rent, or buy houses and apartments anywhere in Nepal.

---

## ✨ Features

- 🔍 **Search & Filter** – Find homes by location, price range, or type  
- 📍 **Map Integration** – View nearby properties on an interactive map  
- 🏡 **Add Listings** – Owners can post their properties with images and details  
- ❤️ **Favorites** – Save your favorite homes for quick access  
- 👤 **Secure Authentication** – Signup and login with OTP verification  
- 📸 **Image Uploads** – Upload property photos easily using Cloud Storage  

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-------------|
| **Frontend** | Flutter |
| **Backend** | Node.js + Express |
| **Database** | Postgresql |
| **Authentication** | Email + OTP Verification |
| **Version Control** | Git + GitHub |
